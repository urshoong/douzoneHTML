1. 기본 속성 선택자

기본선택자뒤에 [] 를 사용하여 속성과 속성값을 사용한다.

        선택자[속성=속성값]{
            스타일속성: 값
        }

2.자손선택자와 후손 선택자

자손: 바로 하위요소들/ 후손: 하위요소들 전부

        * 자손선택자 
        a>b{
            스타일속성:값;
        }
        * 후손선택자 
        a b{
            스타일속성:값;
        }

3.동위선택자

동위관계(같은레벨) 에서 뒤에 위치한 태그를 선택할때 사용한다.

        * a뒤에있는 b요소 하나만선택
        a+b{
            스타일속성:값;
        }

        * a뒤에있는 모든 b 요소 선택
        a~b{
            스타일속성:값;
        }

4.반응선택자

사용자의 움직임에따라 달라지는 선택자

        * 사용자가 클릭한태그
        선택자:active{
            스타일속성:값;
        }

        * 사용자의 마우스가 올라가 있는 태그
        선택자:hover{
            스타일속성:값;
        }

5.상태선택자

입력양식의 상태에따라 선택되는 선택자


        1) 체크된상태의 태그
        선택자:checked{
            스타일속성:값;
        }
    
        2)초점이 맞추어진 input 태그: 선택자:focus{설정내용}
    


        3) 사용가능한 input 태그 선택 :input태그:enabled{설정내용},
           사용불가능한 input 태그 선택 : input태그:disabled{설정내용}
