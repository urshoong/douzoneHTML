font-size속성 : 텍스트의 크기를 변경할때 사용한다.

선택자{ font-size :크기;}형식으로 사용된다.
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 %테스트

font-weight속성 : 두꺼운글씨를 표현 할때 사용한다.

선택자{font-weight : normal | bold | bolder | lighter | 100|400|900; 형식으로 표현}
  • 원래굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게

font-variant속성: 작은 대문자로 변경할 때 사용한다.

선택자 {font-variant:small-caps;} 형식으로 사용

font-style 속성 : 텍스트의 기울임을 지정할때 사용한다.

선택자{font-style : normal | italic | oblique} 의 형식으로 사용
  • italic 기울임 글꼴
  • oblique 기울임 글꼴

font 속성 : 글꼴 관련 스타일을 한번에 지정 할때 사용한다.

 선택자 { font : font-style font-variant font-weight font-size/line-height font-family; }의 형식으로 사용한다.
