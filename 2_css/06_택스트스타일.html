텍스트 스타일

color 속성 : 텍스트의 색상을 지정할 수 있다.

        선택자{
            color:색상명영문이름|16진수숫자|rgb|rgba|hsl|hsla;
        }

색상값 참조사이트 : 사이트로이동

  • 색상명으로 지정
  • 16진수값으로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정

text-decoration 속성 : 텍스트에 줄을 긋거나 줄을 없앨때 사용

        선택자{
            text-decoration:none|underline|overline|line-through;
        }
  • 하이퍼링크 밑줄 없애기
  • 텍스트영역 아래 줄 긋기
  • 텍스트영역 중간에 줄 긋기
  • 텍스트영역 위에 줄 긋기

text-transform 속성 : 영문 텍스트의 대소문자 변환시 사용

        선택자{
            text-transform:none|capitalize|uppercase|lowercase;
        }
  • none
  • capitalize : 영문자의 first 글자만 대문자로
  • uppercase : 모든 영문자를 대문자로
  • Lowercase : 모든 영문자를 소문자로

text-shadow 속성 : 텍스트에 그림자 효과를 줄 때 사용

        선택자{
            text-shadow:none|가로거리 세로거리 번짐정도 색상;
        }
