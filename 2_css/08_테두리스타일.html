        선택자{
            border-style:none|hidden|dotted|dashed|double|groove|inset|outset|ridge|solid;

            border[-위치-]color : 테두리색상;

            border[-위치] : 두께 스타일 색상;

            border[-위치][-위치]radius : 숫자 숫자;

            box-shadow : 수평거리 수직거리 흐림정도 번짐정도 색상
        }
