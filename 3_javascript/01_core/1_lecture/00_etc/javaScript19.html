정규표현식

정규표현식 (REGEX / REGEXP : Regular Expression) 이란 ,
특정규칙을 가진 문자열을 검색하거나 치환할때 사용 하는 형식의 언어

정규 표현식을 이용하면 입력된 문자열에 대해 특정조건을 검색이나 치환시
복잡한 조건을 제시할 필요없이 간단하게 처리가능하다

정규표현식 객체생성

대체문자

정규 표현식을 사용하면 문자열객체를 가지고 replace() 함수를 통해
치환할때 "치환할값" 으로 "대체문자" 를 활용할수있다.

플래그문자

g(global): 전역비교를 수행한다
i(ignoreCase): 대소문자를 가리지않고 비교
m(multiLine):여러줄의 검사를 수행한다

앵커문자

문자열의 앞뒤를 구분해주는 정규표현식 기호

* 메타 문자

특정 메타 문자를 통해 정규식으로 특정 패턴으로 제시가능
따라서 어떤 특정 문자열이 해당 정규식으로 제시한 패턴에 만족하는지
test()를 통해 검사하거나, replace() 이용해서 치환도 가능
