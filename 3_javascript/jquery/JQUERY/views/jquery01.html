제이쿼리 개요

jquery란?

존 레식에 의해 개발된 경량 javascript 라이브러리로 복잡한 javascript코드를
손쉽게 구현하기위해 개발되었다.
jquery의 장점
1. DOM 과 관련된 처리를 쉽게 구현할수 있다
2. ajax 통신, 이벤트, 처리등을 폭넓게 지원한다.
3. 별도의 플러그인 을 통해 차트 , 슬라이스 쇼, 테이블등을 간단히 구현할수 있다

제이쿼리 연결

1. jquery 홈페이지에서 js 파일을 다운받아 연결하는 방법

2. CDN(Content Delevery Network)를 통한 연결

$(document).ready()

페이지를 로드한후 ready 메소드를 실행한다. 자바스크립트의 onload 속성과 같은 개념의 메소드
