이벤트 연결 메소드

mouseenter : 이벤트 대상에 마우스가 올라오면 발생 (하위요소에 영향받지않음)
mouseleave : 이벤트 대상에 마우스가 완전히 벗어나면 발생 (하위요소에 영향받지않음)
mouseout : 이벤트 대상에 마우스가 벗어나면 발생 (하위요소에 영향받음)
mouseover : 마우스 위치가 요소위에 존재하면 발생 (하위요소에 영향 받음 )

mouseover와 mouseenter

키보드이벤트

keydown : 키보드가 눌려질때 keypress : 글자가 입력될때 keyup : 키보드가 떼어질때
